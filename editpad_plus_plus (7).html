How to Use EditPad++

Welcome to EditPad++! This tool helps you generate and copy various special text elements, and now includes powerful AI tools and text utilities.

1. Blank Text Generator

Enter a number in the input field to specify how many "blank" (zero-width space) characters you want to generate. Click "Generate Blank Text" to create them. These characters are invisible but can be useful for formatting or creating seemingly empty lines in some contexts. Click "Copy" to put them on your clipboard.

2. Unsupported Flags & Regional Indicators

This section provides examples of regional indicator symbol sequences that may not render as a single flag emoji on all devices or platforms. Instead, they might appear as two separate letter emojis. You can click the "Copy" button next to each one to copy it to your clipboard and test how it appears on different systems.

3. Emojis & Unicode Characters

  • **Copy Shower Emoji:** Click the large " Copy Shower Emoji" button to quickly copy this specific emoji.
  • **Copy Unlocked Emoji:** Click the large " Copy Unlocked Emoji" button to quickly copy this specific emoji.
  • **Alt Emojis / Niche Emojis:** This grid displays a variety of less common or newer emojis. Click the "Copy" button below each emoji to add it to your clipboard.
  • **Random Unicode Character:** Enter a number (1-20) to generate that many random Unicode characters. These can include a wide range of symbols, scripts, and lesser-known characters. Click "Generate Unicode" and then "Copy" to use them.
  • **Emoji Search:** Type keywords (e.g., "cat", "happy") into the search bar and press Enter or click "Search" to find relevant emojis.
  • **Unicode Character Info:** Enter a single character, and click "Get Info" to see its Unicode code point and name.

4. AI Tools (Powered by Gemini)

  • **Summarize Text:** Paste or type text into the input area and click "Summarize" to get a concise overview.
  • **Explain Characters:** Enter characters or emojis, and click "Explain" to get a description from Gemini.
  • **Creative Writer:** Provide a prompt (e.g., "Write a poem about a rainy day") and click "Generate" to get creative text.
  • **Sentiment Analysis:** Input text and click "Analyze Sentiment" to determine if it's positive, negative, or neutral.
  • **Grammar & Spelling Check:** Enter text and click "Check Grammar" for corrections and explanations.
  • **Keyword Extraction:** Paste text and click "Extract Keywords" to identify important terms.

5. Text Utilities

  • **Text Reverser:** Type or paste text, and it will be instantly reversed.
  • **Case Converter:** Convert text to UPPERCASE, lowercase, Title Case, or Sentence case.
  • **Word & Character Counter:** Type in the text area to see live counts of words and characters.

6. Dark Mode Toggle

Located in the header, this button allows you to switch between light and dark themes for the website. Your preference is saved in your browser's local storage.

7. Windows Style Toggles (1.0, 3.1, 95/98, Me/2000, XP, XP Google, 11, 12)

Explore various Windows operating system aesthetics! Each button in the header applies a distinct visual theme. Activating any Windows style will automatically disable other Windows styles and set the base theme to light mode. Switching back to "Light Mode" or "Dark Mode" will deactivate any active Windows style.

  • **Win 1.0 Style:** The classic grayscale, sharp borders, and pixelated fonts of the very first Windows.
  • **Win 3.1 Style:** The flat, blocky look with a distinct blue title bar.
  • **Win 95/98 Style:** The chunky, gray, and sharp-edged aesthetic of the late 90s.
  • **Win ME/2000 Style:** A slightly softer, more refined retro look.
  • **Win XP Style:** The iconic blue/green gradients and rounded elements of Windows XP.
  • **Win XP Google Style:** This mode transforms the application into a simple browser-like interface with search and quick links to wikis, while applying the Google-themed XP style.
  • **Win 11 Style:** A modern, clean look with subtle shadows and more rounded corners.
  • **Win 12 Style:** A hypothetical future Windows design emphasizing minimalism, soft gradients, and sleekness.

8. Relaxation & Music

The "Settings" button in the header opens a panel where you can find a relaxing video and control background music. You can play/pause the music and adjust its volume (100%, 50%, 1%).

9. Privacy Policy & Terms of Service

Access the Privacy Policy and Terms of Service via links in the footer.

10. Gemini Button

Click the "Gemini" button in the footer to open the Gemini website with a classic "beep" sound.

11. ChatGPT Button

Click the "ChatGPT" button in the footer to open the ChatGPT website with a classic "beep" sound.

12. Download HTML File

The "Download HTML File" buttons (in the header and footer) allow you to save this entire web page to your computer. Once downloaded, you can open the `.html` file directly in any web browser to use EditPad++ offline.

13. Download App

The "Download App" buttons (in the header and footer) also allow you to save this entire web page to your computer as an HTML file. This is effectively the same as "Download HTML File" but provides a more intuitive label for users who perceive the web page as an application.

14. Download as Folder (with limitation)

The ability to directly download the application as a folder (e.g., a ZIP file containing multiple files) is not supported due to browser security restrictions. For now, please use the "Download HTML File" or "Download App" options. If you wish to have it in a folder, you can manually create a folder on your computer and place the downloaded HTML file inside it.

Enjoy using EditPad++!
